<Steps>
  <Step title="Install the package">
    <CodeGroup>

    ```bash npm
    npm install solid-query-state
    ```

    ```bash pnpm
    pnpm add solid-query-state
    ```

    ```bash yarn
    yarn add solid-query-state
    ```

    ```bash bun
    bun add solid-query-state
    ```

    </CodeGroup>
  </Step>

  <Step title="Wrap your app with QueryStateAdapter">
    Add `QueryStateAdapter` at the root of your component tree. This sets up the URL listener that all hooks depend on.

    ```tsx src/index.tsx
    import { render } from "solid-js/web";
    import { QueryStateAdapter } from "solid-query-state";
    import App from "./App";

    render(
        () => (
            <QueryStateAdapter>
                <App />
            </QueryStateAdapter>
        ),
        document.getElementById("root")!,
    );
    ```
  </Step>

  <Step title="Sync a single value to the URL">
    Use `createQueryState` like `createSignal`. Pass a key (the URL param name) and a parser that defines the type.

    ```tsx
    import { createQueryState, parseAsInteger } from "solid-query-state";

    function Counter() {
        const [count, setCount] = createQueryState("count", parseAsInteger.withDefault(0));

        return (
            <div>
                <p>Count: {count()}</p>
                <button onClick={() => setCount((c) => c + 1)}>+</button>
                <button onClick={() => setCount((c) => c - 1)}>-</button>
            </div>
        );
    }
    ```

    Visit `?count=5` and the counter starts at 5. Clicking the buttons updates the URL in real time.
  </Step>

  <Step title="Sync multiple values at once">
    Use `createQueryStates` to manage several URL params with a single getter and setter.

    ```tsx
    import { createQueryStates, parseAsString, parseAsInteger } from "solid-query-state";

    function Filters() {
        const [filters, setFilters] = createQueryStates({
            search: parseAsString.withDefault(""),
            page: parseAsInteger.withDefault(0),
        });

        return (
            <div>
                <input
                    value={filters().search}
                    onInput={(e) => setFilters({ search: e.currentTarget.value })}
                />
                <button onClick={() => setFilters({ page: filters().page + 1 })}>
                    Next page
                </button>
                <button onClick={() => setFilters(null)}>Reset all</button>
            </div>
        );
    }
    ```

    Each key maps to its own URL parameter. Updating one key does not affect the others.
  </Step>
</Steps>
